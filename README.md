# Formula 1 Technical Analysis Portfolio

Portfolio of technical analyses developed as **F1 Technical Writer and Data Analyst (FIA Accredited)** for **F1inGenerale**.

Since 2021, I have been producing engineering-focused analyses covering Formula 1 events through telemetry interpretation, vehicle performance evaluation and technical investigations of race cars.

My work is supported by **custom-developed Python telemetry tools**, enabling engineering analyses both during live sessions and post-session reviews.

---

## Main activities

- Live telemetry and lap time analysis
- Driver and car performance comparison
- Analysis of vehicle setup, power unit deployment and performance evolution throughout race weekends and individual sessions
- Race strategy evaluation
- Technical analysis of aerodynamic developments, suspension systems and steering wheels
- FIA-accredited paddock coverage during Formula 1 and WEC events

---

## Custom telemetry analysis tools

To support my technical analyses, I developed custom telemetry analysis software based on Python. The graphical interface was developed in collaboration with a colleague in Computer Science using HTML, CSS and JavaScript.

The software consists of two complementary modules.

### Live Telemetry Interface

- Live telemetry monitoring for all drivers
- Driver and car performance comparison
- Speed, throttle, brake, gear and RPM analysis
- Automatic speed peak detection
- Absolute and rolling top speed monitoring
- Tyre compound, stint and lap time tracking
- Engineering visualizations for live race analysis

## Future developments

The software is currently being expanded with a dedicated **Race Pace Analysis** interface.

The objective is to centralize stint performance, tyre degradation and lap time evolution within a unified analysis environment, providing a robust data framework for future predictive race pace and performance models.
